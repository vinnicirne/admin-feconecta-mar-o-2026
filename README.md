# FeConecta Admin

Painel administrativo do FeConecta preparado para Vercel + Supabase, com fronteiras claras para crescer com workers e servicos especializados no futuro.

## Stack

- Next.js App Router
- Supabase Auth, Postgres, Realtime e Storage
- Vercel para frontend e BFF leve
- Servicos dedicados futuros para moderacao, billing e notificacoes

## Rodando localmente

```bash
npm install
npm run dev
```

Copie `.env.example` para `.env.local` e configure as chaves do Supabase.
